# ⚡ 湖北风电光伏 D+3 现货交易实战沙盘

> **本项目是一个专为湖北省光伏新能源交易员打造的 D+3 时点级滚搓交易量化辅助系统。**

## 🎯 系统核心定位
在电力现货 D+3 市场中，交易员面临着**“预测误差风险”、“单边偏差重罚”以及“市场流动性不足”**的三重压力。本沙盘系统旨在通过量化算法，将全月的超缺额考核压力分摊至日内的 24 小时，并给出最优的买卖动作与抢单指导价，帮助操盘手实现**“安全水位内的利润最大化”**。

## 💡 核心功能特性

* **🚀 极速无缝演算引擎**：突破 Streamlit 框架局限，实现底层状态与前端表格的实时无刷新绑定。修改预测电量或底仓数据，超缺额指标**零延迟瞬间重算**，告别页面闪退与滚动条跳跃。
* **🛡️ 双向底线防守体系**：
    * **容偏红线监控**：严格控制 $\pm 5\%$ 的单日结算免罚区间。
    * **超缺额强制斩仓**：实时监测 `累计上网量系数` 与 `净合约系数` 的硬性约束（如 0.8 / 0.9 / 1.1），一旦触警，系统将无视一切现货价差，强制输出反向平仓指令。
* **⚔️ 实战级买卖盘口博弈**：
    * **套利摩擦死区**：过滤无效的微小价差波动，避免频繁交易产生的手续费与预测反转风险。
    * **抢单确定性溢价 (Price Buffer)**：在必须买入补仓时，系统自动在现货预测价上附加“溢价小费”，确保在极限行情下能扫到余电，对冲更高昂的偏差罚款。
    * **盘面流动性深度截断**：模拟真实市场深度，当单时点策略需求量大于市场可买卖总量时，系统强制截断，倒逼交易员进行跨时点的“提前建仓”与动态规划。

## 📊 操盘手驾驶舱模块
系统顶部配备了全局战略面板，交易员可一眼获悉：
1. 全天总计防守补仓量 (买入)
2. 全天总计主动套利量 (卖出)
3. 全天最具风险的**极限止损指导价**及发生时点
4. 市场流动性触顶告警次数

---
*注：本沙盘系统使用的量纲已全部统一为省级交易中心标准的 **元/MWh**。*
