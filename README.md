# ⚡ 湖北风电光伏 D+3 日滚动交易实战沙盘

本项目是一个专为湖北省光伏与风电新能源交易员打造的 **D+3 时点级日滚搓交易量化辅助系统**。

## 🎯 系统核心定位

在电力现货 D+3 日滚市场中，发电侧交易员面临着“预测误差风险”、“单边偏差重罚”以及“市场流动性不足”**的三重压力。本沙盘系统旨在通过量化算法，将全月的超缺额考核压力科学分摊至日内的 24 个独立交易时点，并结合实时盘面价差给出**最优的买卖动作**与**抢单指导价，帮助操盘手实现“安全水位内的利润最大化”。

## 💡 核心功能特性

### 1. 🚀 时点级独立演算与 TWAP 平滑滴灌

* **尊重时点独立性**：严格按照现货市场 24 小时独立商品、独立结算的规则。系统将月度超缺额按剩余天数精准均摊至特定的对应时点（如仅在每天 01:00 填平 01:00 的坑），而非的全天平摊，高度契合真实业务场景。
* **极速无缝演算**：底层状态与前端表格实时无刷新绑定。修改任何预测电量或底仓数据，全盘超缺额指标零延迟瞬间重算。

### 2. 🛡️ 双向底线防守体系

* **容偏红线监控**：严格监控单日结算水位，实时计算最终偏差率。
* **超缺额强制斩仓**：实时监测 `累计上网量系数 (默认0.8)` 与 `净合约超缺额系数 (默认0.9/1.1)` 的硬性约束。一旦触警单日欠发违约，系统将无视一切现货价差，强制输出保底买入指令进行止损。（光伏系数0.8，风电0.9）。

### 3. ⚔️ 实战级买卖盘口博弈引擎

* **收益 PK 终极罗盘**：每操作 1 MWh，系统会自动在“现货套利差价”与“免除考核的影子收益”之间进行真金白银的算账博弈，决定是满额套利、保底止损还是持仓观望。
* **套利摩擦死区 (Friction Margin)**：过滤无效的微小价差波动，避免频繁交易产生的高昂手续费与预测反转风险。
* **抢单确定性溢价 (Price Buffer)**：在必须买入止损补仓时，系统自动在现货预测价上附加缓冲差价，确保在极限行情下能扫到余电。
* **盘面流动性深度截断**：模拟真实市场深度，当单时点策略需求量大于设定的市场最大盘口（如 38 MWh）时，系统强制截断，倒逼交易员进行跨时点提前建仓。

## 📊 操盘手决策驾驶舱

系统顶部配备了全局战略与财务 PnL 面板，核心指标一览无余：

* **全天战略汇总**：
* 总计防守补仓买入量 (MWh) & 主动套利卖出量 (MWh)
* 全天最具风险的极限止损指导价及触警时点
* 全天度电均价 (元/MWh) 实时跟踪
* 市场流动性触顶告警次数


* **五大财务测算指标 (PnL)**：
* 干预前总收益（基准：D+3 不操作的躺平收益）
* **干预后总收益**（操作后的真实总营收及净提升对比）
* 免考核收益（通过 D+3 交易挽回的偏差罚款）
* D+3 总收益及出价范围界限



## 📈 全要素可视化图表

系统提供 4 张联动的数据面板，直观呈现复盘推演结果：

1. **量价预测与基差空间**：24小时预测电量、现货价与 D+4 均价的叠加折线柱状图。
2. **D+3 执行单量监控**：红绿分明的买卖双向柱状图，直观展示流动性触顶截断情况。
3. **水库对冲监控视图**：对比干预前后的偏差水位率，直观呈现策略平滑效果（曲线越平稳，策略越优）。
4. **仓位上网与超缺额走势**：D+3 交易落地前后，合约、实际上网与超缺额数据的演变轨迹。

## 📝 台账与数据说明

* 底部提供完整的 24 小时 D+3 台账明细，利用 Pandas Styling 实现了红绿双色的视觉高亮，方便核对每一个时点的策略判定逻辑（如：`【套利执行】`、`【强制止损】`、`【成本熔断】`、`【平滑调仓】` 等）。
* **注：本沙盘系统使用的量纲已全部统一为省级交易中心标准的 元/MWh 和 MWh。**

## 💻 快速启动

1. 克隆本项目到本地。
2. 安装依赖库：
```bash
pip install streamlit pandas plotly numpy

```


3. 在终端中运行 Streamlit 节点：
```bash
streamlit run app.py

```


4. 系统将在浏览器中自动打开（默认地址 `http://localhost:8501`）。
